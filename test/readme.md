## 几类文件夹的命名规范：
pin_cell是认为基准题（CASMO-4）的功率密度是以燃料棒为范围的，里面分别含有粒子数为5000和200000的算例
pin_powden和pin_power是认为基准题（CASMO-4）的功率密度是以栅元为范围的，并分别含有粒子数为5000和200000的算例
    pin_powden是指功率密度的单位为 MW/kgHM（每千克可裂变核素质量的功率）

    pin_power是指把单位 MW/kgHM（每千克可裂变核素质量的功率）换算成了kW/L（每升栅元总体积的功率）
        在此算例集中，默认轴向高度为1cm，不额外设置hact等


% 基础参数:
% 栅元（blanket）平均功率密度: 79.42 kW/L = 79.42 W/cm³
% 栅元总面积: 2.25 cm²
% 燃料芯块面积: 0.6782 cm²
% HM质量密度: 7.3986 g/cm³
% 每cm高度HM质量: 5.017 g/cm

% --- 方案A: set powden (比功率, MW/kgHM)
% 计算: P = 79.42 W/cm³ * 2.25 cm² = 178.695 W/cm
%       m_HM = 7.3986 g/cm³ * 0.6782 cm² = 5.017 g/cm = 0.005017 kg/cm
%       powden = (178.695/1000) kW/cm / 0.005017 kg/cm = 35.62 kW/kgHM = 0.03562 MW/kgHM

% set powden 0.03562

% --- 方案B: set power (总功率, W)
% 需要知道活性区高度。假设 H = 100 cm:
% P_total = 79.42 W/cm³ * 2.25 cm² * 100 cm = 17869.5 W

% 取消注释以下行以使用方案B（需注释掉方案A）:
% set power 17869.5
