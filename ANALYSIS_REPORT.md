## 2. A组总结

**共性特征:**
- A001-A003 (0%Th): 初始keff高(1.25-1.45)，但末端keff下降至0.80-0.97
- A004-A006 (30%Th): 初始临界裕量适中(0.99-1.27)，FIR达0.70-0.98
- A007 (60%Th, 2%U235): 初始次临界(keff=0.697)，末端恢复至0.845，**FIR=1.38，高转化比表现优异**
- A008-A009 (60%Th): 高Th含量下FIR > 0.9，但初始keff偏低
- A010-A012 (90%Th): FIR极高(1.99-4.02)，但初始keff严重不足(0.24-0.47)

**优势:**
- A007/A008/A009/A012均实现在燃耗过程中维持临界
- A007(60%Th) FIR=1.38，是A组中转化比最高的算例之一
- 末端反应性损失相对较小

**短板:**
- 高Th(≥60%)时初始keff严重不足，需补偿措施
- A007初始keff仅0.697，需额外临界补偿

**高转化比目标评估:**
- A007 (Th=60%, U235=2%): FIR=1.3780，**A组最高转化比算例**
- A004 (Th=30%, U235=2%): FIR=0.9752，次优选择

## 3. B组总结

**共性特征:**
- 全部12个算例初始keff极高(1.57-2.10)
- 末端反应性全部为正(+2871~+50458 pcm)，**严重超临界风险**
- FIR极低(0.06-0.56)，属于消耗型燃料
- U233消耗驱动反应性下降，但末端仍保持高keff

**优势:**
- 初始临界裕量充足
- Pu239/Am241累积极低（毒物可控）

**短板:**
- CR极低，不适合高转化比运行
- 末端超临界，需控制棒或可燃毒物补偿

**高转化比目标评估:**
- B001 (U233=5%): FIR=0.56最高，但仍远低于1.0
- B组不适合"高转化比"目标

**高转化比目标评估:**

## 5. D组总结

**共性特征:**
- 27个算例覆盖Th=70-90%, U235=2-5%, Pu=4-8%
- 初始keff适中(1.35-1.48)
- FIR范围0.42-0.65
- 末端临界状态多样化: 4%Pu组末端keff<1, 6-8%Pu组末端keff>1

**关键发现:**
1. **Th_level↑ → CR_final↑**: Th=90%时CR_final达0.95，Th=70%时仅0.60
2. **Pu_level↑ → keff_final↑**: Pu=8%时keff_final>1.05, Pu=4%时keff_final<0.99
3. **U235_enr↑ → keff_final↑**: 影响较小但稳定
4. **D005, D006, D015, D025-D027**: 燃耗末端接近临界(k≈1.0)，最适合长期运行

**优势:**
- D025-D027 (Th=90%, Pu=8%): keff_final≈1.06, 可维持临界
- D019-D021 (Th=90%, Pu=4%): FIR=0.64, CR_final=0.95, 平衡型配置

**短板:**
- Pu累积随Pu_level↑而↑
- Am241累积: 1.17e+03 ~ 2.82e+03 g/cm³

**高转化比目标评估:**
- D019 (Th=90%, U235=2%, Pu=4%): FIR=0.648, CR_final=0.955, **综合最优**

## 6. 全局优化方向

### 6.1 指标冲突分析

| 目标 | 冲突指标 | 权衡关系 |
|------|----------|----------|
| 高FIR | 初始keff | Th≥60%时FIR>0.9但keff_init<0.5 |
| 末端临界 | 毒物累积 | Pu装料提高末端keff但增加Pu239/Am241 |
| 高转化比 | U233消耗 | 高U233初始装料提高初始keff但降低CR |

### 6.2 推荐优化方向

1. **Th_level优化**: 推荐60-90%范围
   - 依据: A004(30%Th)时FIR=0.98，A007(60%Th)时FIR=1.38，A008(60%Th)时FIR=1.13
   - 但需结合Pu或U235补偿初始keff

2. **Pu_level优化**: 推荐4-6%范围
   - 依据: D001-D018数据显示Pu=4%时末端临界裕量不足，Pu=8%时过临界
   - D005/D006(6%Pu)可实现末端keff≈1.0

3. **U235_enr优化**: 2-3%范围最优
   - 依据: U235变化对FIR影响较小，但对初始keff有正贡献

4. **关键核素控制**:
   - Am241: A组(低Th)最低，B组次之，C/D组较高
   - Sm149: B组累积最高，需关注

## 8. 适合做敏感性分析的变量排序

### 优先级排序（基于对FIR和末端keff影响程度）

| 优先级 | 参数 | 敏感程度 | 建议范围 | 依据 |
|--------|------|----------|----------|------|
| **P1** | Th_level | 极高 | 0.70-0.90 | 决定U233产生量，直接影响CR |
| **P2** | Pu_level | 极高 | 0.04-0.08 | 控制末端临界状态 |
| **P3** | U235_enr | 中 | 0.02-0.05 | 影响初始keff但对FIR影响小 |
| **P4** | U233_enr (B组) | 高 | 0.05-0.15 | 仅B组需考虑 |
| **P5** | 燃耗深度 | 高 | 30-50 MWd/kgHM | 影响毒物累积和CR演化 |

### 敏感性分析设计建议

```
因子设计:
- Th_level: 0.70, 0.80, 0.90 (3水平)

→ 共27个组合，与现有D组算例重合

不确定性分析重点:

```
